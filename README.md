git_tutorial

## 1차 ##
1. A가 깃허브 레파지토리 생성
2. B가 해당 레파지토리를 clone 후 각자 branch 생성
3. A가 index.html 파일을 생성 후 add, commit, push, pull request를 함
4. B가 pull request를 확인하고 merge를 함
5. B가 index.html 파일을 수정하고 add, commit, push, pull request를 함
6. A가 pull request를 확인하고 merge를 함

## 2차 ##
1. A가 두번째 파일 index.css 파일을 생성
2. B가 두번째 파일 main.html을 생성
3. A가 index.css파일을 add, commit, pull request를 하고 merge까지 함
4. B가 메인 원격저장소를 fetch하고 main.html을 add, commit, pull request함
5. A가 확인후 merge
